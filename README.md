# HAO XINLEI | AI Creative Portfolio

郝芯蕾的静态单页 AI 创意作品集，展示：

- Gesture Camera：基于 MediaPipe 的在线手势交互网页应用
- AI Fashion Design System：基于提示词的 AI 服装设计系统案例研究

## Run Locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173/`.

## Deploy

This is a fully static site. Deploy the folder to Cloudflare Pages, Vercel, Netlify, or any static host.
